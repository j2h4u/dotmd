<objective>
Add a global extraction cache (compound cache_key → entities/CO_OCCURS relations JSON)
in index.db so that GLiNER batch inference is skipped for chunks whose text + model +
entity_types + threshold have already been extracted — surviving file moves and
model-unchanged re-index cycles.

Purpose: GLiNER is the single most expensive per-chunk operation (~4s/chunk on Xeon E3 V2).
After a file move: the chunk is purged and re-discovered as "new", but its text is unchanged.
The extraction cache allows skipping GLiNER inference when cache_key matches.

Cache key = blake2b(raw_chunk_text + model_name + entity_types_hash + threshold_str).
Compound so that a model upgrade, entity_types change, or threshold change automatically
produces cache misses (stale results impossible).

**ExtractionCache class:**

```python
class ExtractionCache:
    """Per-chunk extraction cache keyed on (chunk_text, model_name, entity_types, threshold).

    Stores only chunk-id-independent data:
    - entities_json: [{name, type, source}] — no chunk_ids field
    - co_occurs_json: [{source_id: entity_name, target_id: entity_name,
                        relation_type: "CO_OCCURS", weight: 1.0}]

    MENTIONS relations (source_id = chunk_id) are NEVER stored.
    They are rebuilt at read time using the current chunk.chunk_id.

    Invalidation: full table clear when model_sig changes.
    model_sig = blake2b(model_name + entity_types_hash + str(threshold)).hexdigest()
    """
```

**ExtractionCache.__init__(self, conn, model_name: str, entity_types: list[str], threshold: float):**
- Store conn, model_name, threshold.
- `self._entity_types_hash = hashlib.blake2b(",".join(sorted(entity_types)).encode()).hexdigest()`
- `self._model_sig = hashlib.blake2b((model_name + self._entity_types_hash + str(threshold)).encode()).hexdigest()`
- Call `self.ensure_table()` — creates tables only, does NOT write sentinel.

**ExtractionCache.ensure_table() -> None:**
- Execute the two CREATE TABLE IF NOT EXISTS statements above.
- `conn.commit()`
- Does NOT write or read extraction_cache_meta. Table creation is separate from sentinel management.

**ExtractionCache.should_invalidate(self) -> bool:**
- `row = conn.execute("SELECT value FROM extraction_cache_meta WHERE key = 'model_sig'").fetchone()`
- If row is None → return False (first run, no previous state to invalidate).
- return `row[0] != self._model_sig`

**ExtractionCache.clear(self) -> None:**
- `conn.execute("DELETE FROM extraction_cache")`
- `conn.execute("DELETE FROM extraction_cache_meta")`
- `conn.execute("INSERT OR REPLACE INTO extraction_cache_meta VALUES ('model_sig', ?)", (self._model_sig,))`
- `conn.commit()`

**ExtractionCache.update_model_sig(self) -> None:**
- `conn.execute("INSERT OR REPLACE INTO extraction_cache_meta VALUES ('model_sig', ?)", (self._model_sig,))`
- `conn.commit()`

**ExtractionCache._make_key(self, chunk_text: str) -> str:**
- Cache key uses RAW chunk.text (not the pipeline's text_hash which is for enriched text).
- `chunk_text_hash = hashlib.blake2b(chunk_text.encode()).hexdigest()`
- `return hashlib.blake2b((chunk_text_hash + self._model_sig).encode()).hexdigest()`

**ExtractionCache.lookup_batch(self, chunks) -> tuple[dict, list]:**
- Parameter type: list of Chunk objects (use `from __future__ import annotations` + `TYPE_CHECKING`
  guard for the Chunk import to avoid circular imports at runtime).
- Compute cache_key for each chunk via `self._make_key(chunk.text)`.
- Query in batches of 500: `SELECT cache_key, entities_json, co_occurs_json FROM extraction_cache WHERE cache_key IN (?,?,...)`
- For hits: build `hits_dict = {chunk_id: (entities_raw, co_occurs_raw)}` where:
  - `entities_raw = json.loads(entities_json)` → list[dict] with keys {name, type, source}
  - `co_occurs_raw = json.loads(co_occurs_json)` → list[dict]
  - Map cache_key back to chunk_id using the computed key → chunk_id mapping.
- Return `(hits_dict, miss_chunks)` where miss_chunks is the list of Chunk objects not in cache.

**ExtractionCache.store_batch(self, chunks, results_per_chunk: dict) -> None:**
- `results_per_chunk`: `{chunk_id: (entities_raw, co_occurs_raw)}`
  where entities_raw is list[dict] with keys {name, type, source} (NO chunk_ids).
  and co_occurs_raw is list[dict] CO_OCCURS relations only.
- For each chunk: `INSERT OR IGNORE INTO extraction_cache (cache_key, entities_json, co_occurs_json) VALUES (?, ?, ?)`
- Do NOT commit — caller commits.

TYPE_CHECKING guard for Chunk import:

```python
def extract_with_cache(self, chunks: list[Chunk]) -> ExtractionResult:
    """Run NER extraction with cache lookup. Falls back to extract() when cache is None.

    Cache stores only chunk-id-independent data:
    - entities: {name, type, source} — no chunk_ids in stored rows
    - co_occurs: CO_OCCURS relations keyed on entity names

    MENTIONS relations are rebuilt here at read time using current chunk.chunk_id.
    This makes cached results safe to reuse after Plan 03 chunk_id migration.
    """
    if self._extraction_cache is None or not chunks:
        return self.extract(chunks)

    cached_hits, miss_chunks = self._extraction_cache.lookup_batch(chunks)

    # Build a chunk_id lookup for MENTIONS reconstruction
    chunk_by_id: dict[str, Chunk] = {c.chunk_id: c for c in chunks}

    entities: list[Entity] = []
    relations: list[Relation] = []
    seen_entities: dict[tuple[str, str], Entity] = {}

    # Restore cached results and rebuild MENTIONS at read time
    for chunk_id, (ents_data, co_occurs_data) in cached_hits.items():
        chunk = chunk_by_id[chunk_id]
        span_counter: Counter[str] = Counter()

        for d in ents_data:
            key = (d["name"].lower(), d["type"].lower())
            if key not in seen_entities:
                entity = Entity(
                    name=d["name"],
                    type=d["type"],
                    source=d["source"],
                    chunk_ids=[chunk_id],
                )
                seen_entities[key] = entity
                entities.append(entity)
            else:
                existing_entity = seen_entities[key]
                if chunk_id not in existing_entity.chunk_ids:
                    existing_entity.chunk_ids.append(chunk_id)
            span_counter[d["name"]] += 1

        for d in co_occurs_data:
            relations.append(Relation(**d))

        # Rebuild MENTIONS using current chunk_id
        for name, freq in span_counter.items():
            relations.append(Relation(
                source_id=chunk_id,
                target_id=name,
                relation_type="MENTIONS",
                weight=float(freq),
            ))

    # Run GLiNER only on cache misses
    new_results_per_chunk: dict[str, tuple[list, list]] = {}
    if miss_chunks:
        miss_result = self.extract(miss_chunks)

        # Split miss_result into per-chunk data for cache storage
        # Build per-chunk entity/co_occurs from the batch result
        # We re-run extract() which returns a combined ExtractionResult —
        # merge its entities/relations into our running result and build
        # cache payload from what extract() returned for miss_chunks.

        # Merge entities from miss (extract() already handles seen_entities dedup
        # at the batch level — we just append here, accepting that cross-chunk
        # dedup is batch-scoped in extract())
        entities.extend(miss_result.entities)

        # Separate MENTIONS from CO_OCCURS for cache storage
        for chunk in miss_chunks:
            chunk_mentions = [
                r for r in miss_result.relations
                if r.relation_type == "MENTIONS" and r.source_id == chunk.chunk_id
            ]
            chunk_co_occurs = [
                r for r in miss_result.relations
                if r.relation_type == "CO_OCCURS"
            ]
            # entities for this chunk (those that mention this chunk_id)
            mentioned_names = {r.target_id for r in chunk_mentions}
            chunk_entities = [
                e for e in miss_result.entities
                if e.name in mentioned_names
            ]

            entities_to_cache = [
                {"name": e.name, "type": e.type, "source": e.source}
                for e in chunk_entities
            ]
            co_occurs_to_cache = [r.model_dump() for r in chunk_co_occurs]
            new_results_per_chunk[chunk.chunk_id] = (entities_to_cache, co_occurs_to_cache)

        relations.extend(miss_result.relations)

        if new_results_per_chunk:
            self._extraction_cache.store_batch(miss_chunks, new_results_per_chunk)

    return ExtractionResult(entities=entities, relations=relations)
```
