<objective>
Add a global extraction cache (compound cache_key → entities/relations JSON) in index.db
so that GLiNER batch inference is skipped for chunks whose text+model+entity_types have
already been extracted — surviving file moves and model-unchanged re-index cycles.

Purpose: GLiNER is the single most expensive per-chunk operation (~4s/chunk on Xeon E3 V2).
After a file move: the chunk is purged and re-discovered as "new", but its text is unchanged.
The extraction cache allows skipping GLiNER inference when cache_key matches.

Cache key = blake2b(text_hash + model_name + entity_types_hash) — compound so that a model
upgrade or entity_types change automatically produces cache misses (stale results impossible).

Output: ExtractionCache added to storage/cache.py; NERExtractor modified to accept an
optional ExtractionCache and use it in extract(); pipeline.__init__() wires invalidation
logic and passes cache to NERExtractor.
</objective>

ExtractionCache class:

```python
class ExtractionCache:
    """Per-chunk extraction result cache keyed on (text, model, entity_types).

    Cache key = blake2b(chunk_text_hash + model_name + entity_types_hash).
    Full table clear is performed when model_name or entity_types change.
    """
```

ExtractionCache.__init__(self, conn: sqlite3.Connection, model_name: str, entity_types: list[str]):
- Store conn, model_name.
- Compute: self._entity_types_hash = hashlib.blake2b(",".join(sorted(entity_types)).encode()).hexdigest()
- Call self.ensure_table().

ExtractionCache.ensure_table() -> None:
```sql
CREATE TABLE IF NOT EXISTS extraction_cache (
    cache_key TEXT PRIMARY KEY,
    entities JSON NOT NULL,
    relations JSON NOT NULL,
    created_at TEXT NOT NULL DEFAULT (datetime('now'))
)
```
Then conn.commit().

ExtractionCache._make_key(self, chunk_text: str) -> str:
- chunk_text_hash = hashlib.blake2b(chunk_text.encode()).hexdigest()
- return hashlib.blake2b((chunk_text_hash + self._model_name + self._entity_types_hash).encode()).hexdigest()

ExtractionCache.should_invalidate(self) -> bool:
- Query: SELECT cache_key FROM extraction_cache LIMIT 1
- If table is empty → return False (nothing to invalidate)
- Query: SELECT entities FROM extraction_cache LIMIT 1 → peek at any row to get the stored model signature
  Actually simpler: store the model signature as a sentinel row in a separate small table:
  
  Add a second table in ensure_table():
  ```sql
  CREATE TABLE IF NOT EXISTS extraction_cache_meta (
      key TEXT PRIMARY KEY,
      value TEXT NOT NULL
  )
  ```
  Then upsert current signature: INSERT OR REPLACE INTO extraction_cache_meta VALUES ('model_sig', ?)
  where model_sig = blake2b(model_name + entity_types_hash).hexdigest()
  
  should_invalidate() reads the stored model_sig and compares to current:
  ```python
  current_sig = hashlib.blake2b(
      (self._model_name + self._entity_types_hash).encode()
  ).hexdigest()
  row = conn.execute(
      "SELECT value FROM extraction_cache_meta WHERE key = 'model_sig'"
  ).fetchone()
  if row is None:
      return False  # first run, nothing stored yet
  return row[0] != current_sig
  ```

ExtractionCache.clear(self) -> None:
- DELETE FROM extraction_cache
- DELETE FROM extraction_cache_meta
- Then upsert the current model_sig into extraction_cache_meta.
- conn.commit()

ExtractionCache.update_model_sig(self) -> None:
- Upsert current model_sig into extraction_cache_meta.
- conn.commit()

ExtractionCache.lookup_batch(self, chunks: list) -> tuple[dict[str, tuple[list, list]], list]:
- Takes list[Chunk] (use Any import or import Chunk from dotmd.core.models — use TYPE_CHECKING guard to avoid circular imports).
- Compute cache_key for each chunk.
- Single SELECT IN query for all keys (batch of 500 max).
- Return (hits_dict, miss_chunks):
  - hits_dict: {chunk_id: (entities_data, relations_data)} where entities_data/relations_data are raw list[dict] from JSON (not yet Pydantic objects — caller reconstructs).
  - miss_chunks: list of Chunk objects not found in cache.

ExtractionCache.store_batch(self, chunks: list, results_per_chunk: dict[str, tuple[list, list]]) -> None:
- results_per_chunk: {chunk_id: (entities_list[dict], relations_list[dict])}
- For each chunk in chunks: INSERT OR IGNORE with json.dumps(entities), json.dumps(relations).
- Do NOT commit — caller commits.

Serialization/deserialization contract:
- Store: json.dumps([e.model_dump() for e in entities]) — Pydantic v2 model_dump()
- Retrieve: raw list[dict] returned to caller; caller does [Entity(**d) for d in data]

3. Modify extract() to add cache lookup before GLiNER and store after:

Replace the current body of extract() (from `model = self._get_model()` to the return statement) with:

```python
if not chunks:
    return ExtractionResult()

# --- Cache lookup ---
if self._extraction_cache is not None:
    cached_hits, miss_chunks = self._extraction_cache.lookup_batch(chunks)
else:
    cached_hits, miss_chunks = {}, list(chunks)

entities: list[Entity] = []
relations: list[Relation] = []
seen_entities: dict[tuple[str, str], Entity] = {}

# Restore cached results
for chunk_id, (ents_data, rels_data) in cached_hits.items():
    for d in ents_data:
        key = (d["name"].lower(), d["type"].lower())
        if key not in seen_entities:
            entity = Entity(**d)
            seen_entities[key] = entity
            entities.append(entity)
    for d in rels_data:
        relations.append(Relation(**d))

# Run GLiNER only on cache misses
new_results_per_chunk: dict[str, tuple[list, list]] = {}
if miss_chunks:
    model = self._get_model()
    texts = [chunk.text for chunk in miss_chunks]
    batch_predictions: list[list[dict[str, Any]]] = model.inference(
        texts, self._entity_types, threshold=self._threshold,
    )

    for chunk, predictions in zip(miss_chunks, batch_predictions):
        chunk_entities_data: list[dict] = []
        chunk_relations_data: list[dict] = []

        if not predictions:
            new_results_per_chunk[chunk.chunk_id] = ([], [])
            continue

        span_counter: Counter[str] = Counter()
        chunk_entity_keys: list[tuple[str, str]] = []

        for pred in predictions:
            name: str = pred["text"].strip()
            etype: str = pred["label"]
            if not name:
                continue

            span_counter[name] += 1
            key = (name.lower(), etype.lower())
            chunk_entity_keys.append(key)

            if key not in seen_entities:
                entity = Entity(
                    name=name,
                    type=etype,
                    source=ExtractDepth.NER,
                    chunk_ids=[chunk.chunk_id],
                )
                seen_entities[key] = entity
                entities.append(entity)
                chunk_entities_data.append(entity.model_dump())
            else:
                existing = seen_entities[key]
                if chunk.chunk_id not in existing.chunk_ids:
                    existing.chunk_ids.append(chunk.chunk_id)
                chunk_entities_data.append(existing.model_dump())

        unique_keys = list(dict.fromkeys(chunk_entity_keys))

        for key_a, key_b in combinations(unique_keys, 2):
            rel = Relation(
                source_id=seen_entities[key_a].name,
                target_id=seen_entities[key_b].name,
                relation_type="CO_OCCURS",
                weight=1.0,
            )
            relations.append(rel)
            chunk_relations_data.append(rel.model_dump())

        for key in unique_keys:
            entity = seen_entities[key]
            freq = span_counter[entity.name]
            rel = Relation(
                source_id=chunk.chunk_id,
                target_id=entity.name,
                relation_type="MENTIONS",
                weight=float(freq),
            )
            relations.append(rel)
            chunk_relations_data.append(rel.model_dump())

        new_results_per_chunk[chunk.chunk_id] = (chunk_entities_data, chunk_relations_data)

    # Store new results in cache
    if self._extraction_cache is not None and new_results_per_chunk:
        self._extraction_cache.store_batch(miss_chunks, new_results_per_chunk)

return ExtractionResult(entities=entities, relations=relations)
```
