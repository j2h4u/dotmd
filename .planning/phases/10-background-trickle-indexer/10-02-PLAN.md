    **New function `discover_files_multi(paths: list[str], exclude: list[str]) -> list[FileInfo]`:**

    1. Takes `paths` (list of directory paths or glob patterns) and `exclude` (list of glob patterns to filter out).

    2. For each path_spec in paths:
       - If `path_spec` contains `*` or `?` (it's a glob pattern):
         - Split on first `*` to find the root directory
         - Use `Path(root).glob(pattern)` to find matching files
         - Filter: must be a file, must end with `.md`, must not be excluded
       - Else (it's a directory path):
         - Use `os.walk()` for directory traversal WITH pruning (per Pitfall 7)
         - During walk, prune excluded directory names from `dirs` in-place to avoid traversing node_modules, .git, etc.
         - Collect `.md` files that pass exclude filtering

    3. Deduplication: use a `seen: set[Path]` with resolved paths to avoid duplicates from overlapping patterns.

    4. For each discovered path, create FileInfo using the same pattern as `discover_files()`:
       ```python
       content = read_file(md_path)
       stat = md_path.stat()
       FileInfo(path=md_path, title=_extract_title(content, md_path),
                last_modified=datetime.fromtimestamp(stat.st_mtime, tz=timezone.utc),
                size_bytes=stat.st_size)
       ```
       Wrap in try/except OSError, log warning and skip unreadable files.

    5. Sort results by path for deterministic ordering.

    **Helper `_is_excluded(path: Path, exclude: list[str]) -> bool`:**
    - Check if any path component matches any exclude pattern (stripped of `**/` prefix).
    - Use `fnmatch.fnmatch()` for pattern matching.
    - Example: for exclude pattern `"**/node_modules"`, check if `"node_modules"` appears in `path.parts`.
    - For exclude pattern `"**/.git"`, check if `".git"` appears in `path.parts`.

    **Helper `_prune_dirs(dirs: list[str], exclude: list[str]) -> None`:**
    - Modifies `dirs` list in-place (for os.walk pruning).
    - Remove directory names that match any exclude pattern component.
    - Example: if exclude has `"**/node_modules"`, remove `"node_modules"` from dirs.

    **Imports to add:**
    ```python
    import os
    from fnmatch import fnmatch
    ```

    Per D-08, D-09: glob-based paths + exclude. Per Pitfall 7: prune during traversal for performance.
