## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| live old-stack service -> artifact bundle | Current production-derived baseline results are captured as JSONL evidence. |
| copied/migrated data -> isolated Surreal target | Candidate target is prepared from transform-first evidence without live old-stack mutation. |
| artifact bundle -> operator acceptance | Production-derived diffs inform human acceptance decisions. |
| operator edits -> evaluator | Acceptance ledger rows can alter aggregate gate outcome while preserving raw classifications. |
