## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| sequential-cypher-round-trips | Вызов add_edge/add_entity_node в цикле = 1 HTTP call на операцию. FalkorDB single-threaded per query. 1500 calls × 300ms = 470s/file. | blocking | UNWIND batch: собирать все узлы/рёбра файла, отправлять 3-4 запроса через UNWIND вместо сотен |
| graph-missing-from-schema-wipe | _check_schema_version вайпает vectors/fingerprints/cache но НЕ граф → реиндекс стартует с тысячами старых узлов | blocking | Добавить self._graph_store.delete_all() в _check_schema_version перед запуском реиндекса |
| smoke-tests-call-removed-api | tests/smoke/ вызывают GET /search и GET /status (REST API удалён в 98c6b99). JSONDecodeError при setup. | advisory | Переписать smoke тесты под MCP — см. tests/e2e/ как образец |
