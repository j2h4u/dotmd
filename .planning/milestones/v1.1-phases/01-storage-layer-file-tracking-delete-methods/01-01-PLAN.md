    **Step 3: Create FileTracker**

    Create `backend/src/dotmd/ingestion/file_tracker.py` with the complete FileTracker class and FileDiff dataclass as specified in the research document (RESEARCH.md lines 335-421). Key implementation details:

    - `FileDiff` is a `@dataclass` with four fields: `new: list[str]`, `modified: list[str]`, `deleted: list[str]`, `unchanged: list[str]`, all `default_factory=list`
    - `FileTracker.__init__(self, conn: sqlite3.Connection)` — accepts a connection (shared with metadata.db), creates `file_fingerprints` table via `CREATE TABLE IF NOT EXISTS`
    - Table schema: `file_path TEXT PRIMARY KEY, mtime REAL NOT NULL, size_bytes INTEGER NOT NULL, checksum TEXT NOT NULL, indexed_at TEXT NOT NULL`
    - `diff(self, discovered: list[FileInfo]) -> FileDiff` — two-stage detection:
      1. Load all stored fingerprints into a dict `{path: (mtime, size, checksum)}`
      2. For each discovered file: if not in stored -> new; if mtime+size match -> unchanged (SKIP checksum); if mtime/size differ, compute `hashlib.md5(f.path.read_bytes()).hexdigest()` — if checksum matches stored -> unchanged (update mtime/size in DB); else -> modified
      3. Deleted = stored paths not in discovered set
      - CRITICAL: Do NOT access `FileInfo.checksum` — it's a computed_field that reads file bytes every time. Always compute MD5 explicitly with `hashlib.md5(f.path.read_bytes()).hexdigest()` only when needed.
    - `save_fingerprint(self, file_path: str, mtime: float, size: int, checksum: str) -> None` — INSERT OR REPLACE with `indexed_at = datetime.now(tz=timezone.utc).isoformat()`
    - `remove_fingerprint(self, file_path: str) -> None` — DELETE WHERE file_path = ?
    - `clear(self) -> None` — DELETE FROM file_fingerprints
