**Class `FalkorDBGraphStore`:**

**`__init__(self, url: str = "redis://localhost:6379", graph_name: str = "dotmd") -> None`:**
- Parse `url` with `urlparse` to extract host (default `localhost`) and port (default `6379`)
- Connect: `self._db = FalkorDB(host=host, port=port)`
- Select graph: `self._graph = self._db.select_graph(graph_name)`
- Store `self._graph_name = graph_name` for status reporting
- Create range indexes for performance (idempotent, wrap each in try/except):
  ```python
  for label in ("File", "Section", "Entity", "Tag"):
      try:
          self._graph.query(f"CREATE INDEX FOR (n:{label}) ON (n.id)")
      except Exception:
          pass  # Index already exists
  ```
- If `FalkorDB()` raises `ConnectionError` or `redis.exceptions.ConnectionError`, catch it, log an actionable error message including the URL, and re-raise as `ConnectionError(f"Cannot connect to FalkorDB at {url}. Is the container running?")`

**`add_file_node(self, file_path, title, checksum)`:**
```python
self._graph.query(
    "MERGE (f:File {id: $id}) SET f.title = $title, f.checksum = $checksum",
    params={"id": file_path, "title": title, "checksum": checksum},
)
```

**`add_section_node(self, chunk_id, heading, level, file_path, text_preview)`:**

**`add_entity_node(self, name, entity_type, source)`:**

**`add_tag_node(self, name)`:**

**`add_edge(self, source_id, target_id, relation_type, weight=1.0)`:**
Use label-agnostic MATCH (acceptable for ~3.5K nodes per RESEARCH.md):
```python
self._graph.query(
    "MATCH (a {id: $src}), (b {id: $tgt}) "
    "MERGE (a)-[r:REL]->(b) "
    "SET r.rel_type = $rel_type, r.weight = $weight",
    params={"src": source_id, "tgt": target_id,
            "rel_type": relation_type, "weight": weight},
)
```

**`get_neighbors(self, node_id, max_hops=2)`:**
Use variable-length path with `labels()` (plural — FalkorDB, not `label()` which is LadybugDB):
```python
result = self._graph.ro_query(
    f"MATCH (a {{id: $id}})-[*1..{int(max_hops)}]-(b) "
    "RETURN DISTINCT b.id, labels(b)[0]",
    params={"id": node_id},
)
neighbors = []
for row in result.result_set:
    nid = str(row[0])
    if nid != node_id:
        neighbors.append((nid, "", 1.0))
return neighbors
```

**`get_graph_data(self)`:**
```python
nodes = []
for label in ("File", "Section", "Entity", "Tag"):
    try:
        result = self._graph.ro_query(f"MATCH (n:{label}) RETURN n")
        for row in result.result_set:
            node = row[0]
            node_id = node.properties.get("id", "")
            props = {k: v for k, v in node.properties.items() if k != "id"}
            # Enrich Section nodes with NER entity names
            if label == "Section":
                try:
                    ent_result = self._graph.ro_query(
                        "MATCH (s:Section {id: $sid})-[:REL]->(e:Entity) "
                        "WHERE e.source = 'ner' "
                        "RETURN e.id",
                        params={"sid": str(node_id)},
                    )
                    if ent_result.result_set:
                        props["ner_entities"] = [str(r[0]) for r in ent_result.result_set]
                except Exception:
                    pass
            nodes.append({"id": str(node_id), "label": label, "properties": props})
    except Exception:
        logger.debug("Failed to query %s nodes", label, exc_info=True)

edges = []
try:
    result = self._graph.ro_query(
        "MATCH (a)-[r]->(b) RETURN a.id, b.id, r.rel_type, r.weight"
    )
    for row in result.result_set:
        edges.append({
            "source": str(row[0]),
            "target": str(row[1]),
            "relation_type": str(row[2]),
            "weight": float(row[3]) if row[3] is not None else 1.0,
        })
except Exception:
    logger.debug("Failed to query edges", exc_info=True)

return {"nodes": nodes, "edges": edges}
```
